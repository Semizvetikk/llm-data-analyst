# LLM Data Analyst

## Описание

Приложение для автоматического анализа данных с использованием LLM через API. LLM-агент самостоятельно вызывает необходимые инструменты для анализа загруженного датасета и формирует отчет с ключевыми метриками, корреляциями и инсайтами.

## Особенности

- **Автоматический анализ данных** - LLM самостоятельно определяет, какие инструменты вызвать
- **Реальный анализ, а не генерация текста** - LLM получает данные через вызов функций, а не генерирует на основе описания
- **Защита от prompt-injection** - очистка пользовательского ввода от попыток взлома
- **Безопасное выполнение кода** - встроенные ограничения для предотвращения опасных операций


## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd llm-data-analyst
```
2. Создайте виртуальное окружение:
```python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```
3. Установите зависимости:
```
pip install -r requirements.txt
```
4. Настройте переменные окружения:
- Создайте файл .env в корне проекта
- Добавьте ваш API ключ:
```
OPENROUTER_API_KEY=your_api_key_here
```
## Запуск
```
streamlit run main.py
```
Приложение откроется в браузере по адресу http://localhost:8502
